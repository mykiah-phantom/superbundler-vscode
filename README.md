# SuperBundler VSCode Extension

Bundle any folder into a single HTML file directly from VSCode with options for:

- Custom output file name  
- Minify HTML/CSS/JS  
- Inline images and fonts  
- Auto-open output file  
- Keyboard shortcut (Ctrl+Alt+B)  

## Usage

1. Open Command Palette → `SuperBundler: Bundle Folder`  
2. Pick a folder to bundle  
3. Enter output file name (default `bundle.html`)  
4. Choose minify and image options  
5. The bundled file opens automatically  

## Installation

- Clone repo and `npm install` dependencies  
- Package with `vsce package` → installs `.vsix` file in VSCode  
- Or publish to [VSCode Marketplace](https://marketplace.visualstudio.com/)
